# 통합 변경 요약

## UX/UI
- 모바일 가독성을 기준으로 제목, 본문, 보조문구의 크기와 행간 재조정
- 차분한 임상 서비스 인상을 주는 녹색·아이보리 팔레트 정리
- 버튼 최소 높이 54px, 안전영역 대응, 작은 화면 대응
- 검색 탭·검색 결과·영양 카드·조정 결과 카드의 시각적 위계 개선
- Neon/CSV/오프라인 데이터 상태를 홈에서 표시
- 최초 온보딩 완료 상태를 localStorage에 저장하여 재접속 시 홈으로 이동
- 백엔드 `targets`를 우선 사용해 영양 판정 기준 표시

## 백엔드/DB
- `db_loader.py` 추가
- Neon 데이터를 기존 CSV와 동일한 DataFrame 구조로 변환
- `DATABASE_URL` 설정 시 Neon 우선, 실패 시 CSV 자동 폴백
- `/health`에 `data_source`와 `meal_templates` 추가
- 식단 생성 모델과 레버 조정 알고리즘은 유지하여 회귀 위험 최소화
